# Student Management System - Java

## Description
This project is a student management system developed in Java using a graphical user interface (GUI).

The system allows users to register student data, clear input fields, and display saved student information.

## Features
- Register student information
- Clear input fields
- Display saved student records
- Exit the system
- Implementation of object-oriented programming concepts such as inheritance

## Technologies Used
- Java
- Java Swing (GUI)
- NetBeans GUI Builder
- Object-Oriented Programming (OOP)
